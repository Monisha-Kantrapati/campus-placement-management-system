# Campus Placement Management System

## 🚀 Project Overview
A React-based web application to manage campus placements, allowing students to view jobs, apply, and track applications.

## 🛠 Tech Stack
- React.js
- JavaScript
- HTML, CSS

## ✨ Features
- Student Dashboard
- Job Listings
- Application Tracking
- Responsive Design
